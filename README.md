# Library Management System 

## Description
A console-based Library Management System developed using C++ with Object-Oriented Programming concepts and file handling for persistent data storage.

This project allows users to manage library books efficiently through a simple menu-driven interface. The system supports adding books, displaying records, searching books, issuing books, and returning books. All records are stored in a text file so the data remains saved even after closing the program.

---

## Features
- Add new books
- Display all books
- Search books by title
- Issue books
- Return books
- Persistent storage using file handling
- Menu-driven interface
- Object-Oriented Programming using classes

---

## Technologies Used
- C++
- File Handling
- Object-Oriented Programming (OOP)
- STL Vector

---

## How to Compile

```bash
g++ main.cpp -o library
```

## How to Run
